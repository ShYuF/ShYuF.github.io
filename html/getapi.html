除本接口外，所有可以调用的接口如下：

  1. /test
    • 允许的请求方法：GET, POST
    • url参数：无
    • json参数：无
    • 说明：测试接口，如果连接成功，返回connect successful
  2. /register
    • 允许的请求方法：POST
    • url参数：无
    • json参数：{username : String, password : String}
    • 说明：用户注册，返回状态码
  3. /login
    • 允许的请求方法：POST
    • url参数：无
    • json参数：{username : String, password : String}
    • 说明：用户登录，如果成功，返回用户信息，否则返回错误码
  4. /logout
    • 允许的请求方法：POST
    • url参数：无
    • json参数：{username : String}
    • 说明：退出登录，返回状态码（只有用户处于登录状态时才会正常退出）
  5. /upduser
    • 允许的请求方法：POST
    • url参数：无
    • json参数：{username : String, (可选)nickname : String, (可选)profile : String, (可选)password : String}
    • 说明：更新用户信息，用户必须处于登录状态
  6. /deluser
    • 允许的请求方法：POST
    • url参数：无
    • json参数：{username : String}
    • 说明：删除用户信息，用户必须处于登录状态时才能删除
  7. /save
    • 允许的请求方法：POST
    • url参数：无
    • json参数：{username : String, title : String, (可选)new_title : String, tags : List, content : List}
    • 说明：保存或修改笔记信息（题目、标签、内容等），用户必须处于登录状态时才能保存
  8. /delnote
    • 允许的请求方法：POST
    • url参数：无
    • json参数：{username : String, title : String}
    • 说明：删除笔记，用户必须处于登录状态时才能删除
